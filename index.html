Installation & Setup

Installing Polars in Python

To install Polars, run:

pip install polars

If you want **all dependencies** (CSV, Parquet, JSON support), install:

pip install polars[all]

Installing Polars in Rust

If you're using Rust, add Polars to your project:

[dependencies]
polars = "0.38.0"

Code Examples

Creating a DataFrame

import polars as pl

df = pl.DataFrame({
    "Name": ["Alice", "Bob", "Charlie"],
    "Age": [25, 30, 35],
    "Salary": [50000, 60000, 70000]
})

print(df)

Filtering Data

filtered_df = df.filter(pl.col("Age") > 28)
print(filtered_df)

Grouping & Aggregating

df.groupby("Age").agg(pl.col("Salary").sum())

Reading & Writing Data

df = pl.read_csv("data.csv")
df.write_parquet("output.parquet")
